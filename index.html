Perguntas

2) Responda as perguntas abaixo:
a. Na tag "input" com atributo type="text", para que serve o atributo "placeholder"?
b. Na tag "input" com atributo type="radio" ou type="checkbox", para que serve o atributo "checked"?
c. Na tag "input" para que serve o atributo "value"?
d. Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez?
e. Para que serve a tag "label"? Para que serve o atributo "for" nessa tag?
f. Na tag "option" para que serve o atributo "selected"?
g. Nas tags de formulário para que serve o atributo "required"?
h. Na tag "form" para que serve o atributo "action"?
i. Na tag "form" para que serve o atributo "method"?
j. Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag "form"?
k. Qual valor deve ser informado no atributo "method" na tag "form" em um formulário que será usado para fazer upload de arquivos?


Respostas

2)a. O atributo "placeholder" serve para adicionar o texto entre aspas dentro do quadrante, mas é possível escrever "em cima"

b. O atributo "checked" significa que o quadrante já virá selecionado ou com check

c. O atributo "value" serve para adicionar o texto entre aspas dentro do quadrante, mas não é possível escrever "em cima", o que vem escrito terá de ser apagado

d. O atributo que deve ser utilizado para agrupar vários botões de rádio em um formulário é abrir e fechar o forms dentro da tag fieldset, e para que um por vez seja selecionado é necessário que o "name" seja igual em todos os "input", de forma que caracterizem eles como pertencentes ao mesmo grupo

e. A tag "label" possibilita a escrita em frente ao quandrante de tipo "checkbox" e/ou "radio"; e o atributo "for" serve para conectar o input com o label, ou, o caracter com o quadrante, por isso o ID(do input) e for(do label) devem ser iguais

f. O atributo "selected", na tag "option", define a opção selecionada como já definida primeiro antes mesmo da escolha do usuário

g. O atributo "required", na tag "form", serve para tornar obrigatório a resposta do usuário para a questão

h. O atributo "action", na tag form, é O programa que recebe o código

i. O atributo "method", na tag "form", serve para definir aparecer/esconder a resposta do usuário e retirar ou não o limite de caracter da URL

j. O atributo da tag "method", na tag "form", para que a senha não apareça na URL deve ser "POST"

k. O atributo que será usado para fazer upload de arquivos no "method" deve ser o "POST"
